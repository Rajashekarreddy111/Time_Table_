# NEC Timetable Frontend

Frontend application for the **Narasaraopeta Engineering College Timetable Management System**.

## Tech Stack

- Vite
- React + TypeScript
- Tailwind CSS
- shadcn/ui

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

App runs on `http://localhost:8080` by default.

## Build

```bash
npm run build
npm run preview
```

## Environment

Set backend API base URL in `.env`:

```bash
VITE_API_BASE_URL=http://localhost:5000/api
```
